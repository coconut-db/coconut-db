# CoconutDB

## concepts

### Dictionary
List of all objects, basically a key-value storage of following items:

* Key
* ObjectType
* Create Timestamp
* Last Update Timestamp
* Last Read Timestamp
* Last Commit Timestamp
* Storage Type [Hot, Cold]
* Time To Leave [TTL]

### Object Type
Supported data structures:

* Linear Hash Table
* Sorted Table

### Shards
### Hot and Cold Storage
### Commit and Flush
### Replication
### Leader Election
### Quorum (100% Consensus)
### Voting and Timeout
